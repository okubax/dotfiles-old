# okubax's dotfiles

My dotfiles for:

* bash
* compton
* dunst
* i3-gaps + conkybar
* irssi
* mpd
* mutt
* ncmpcpp
* offlineimap + msmtp + notmuch
* vim
* weechat
* misc. useful & not-so-useful scripts

## Installation

	git clone https://github.com/okubax/dotfiles.git
	cd dotfiles
	./dots.sh
